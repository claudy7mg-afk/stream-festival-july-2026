# Stream Festival

Static festival website for Stream Festival Summer 2026.

## Structure

- `index.html` redirects to the homepage in `src/code.html`
- `src/` contains the site pages
- `config/` contains the page configuration files
- `res/` contains local image assets
- `docs/` contains project documentation

## Pages

- `src/code.html` for the main landing page
- `src/activities.html` for the activities page

## Run

Open `index.html` in a browser, or serve the project root with any static file server.
